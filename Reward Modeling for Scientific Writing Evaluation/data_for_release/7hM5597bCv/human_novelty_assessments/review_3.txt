The paper proposes an offline RL algorithm utilizing latent diffusion skill models for temporal abstraction, and Q learning with these skills. The method is a combination of existing offline RL algorithms (primarily LDCQ and IQL), but there is no proper reason given for this particular configuration of components. The only novel addition seems to be the use of the value function for deciding when to stop executing a skill, but this is a simple iterative improvement. Novelty is not strictly necessary, but the additions made here are not well justified at all with no coherent story surrounding it. Small engineering improvements to boost the score in this benchmark does not give any signal to the true value of the method.
