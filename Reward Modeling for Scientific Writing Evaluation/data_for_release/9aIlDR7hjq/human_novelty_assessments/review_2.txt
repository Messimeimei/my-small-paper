This paper demonstrates that conditioning the generation process on an augmented real image and a text prompt produces effective synthetic datasets, which benefit downstream tasks, particularly for long-tailed (LT) classification and few-shot classification. However, the technical novelty of this paper is unclear. The concept of combining both augmented images and text prompts seems useful for LT and few-shot classification but lacks novelty. If this approach is not technically original, the paper should at least show a broad variety of downstream tasks that benefit from it, which it did not. The contribution is not clearly articulated, as it’s evident that the synthetic dataset is effective, but it’s unclear for which specific tasks it is most useful, with the focus confined to LT and few-shot classification. Expanding the application scenarios would improve the paper’s impact.
