This paper introduces LLM4Solver, a framework that leverages large language models (LLMs) to assist in algorithm design for combinatorial optimization, specifically targeting Mixed Integer Linear Programs. The approach involves using an LLM to generate algorithmic components in a programming language, which are subsequently refined through evolutionary algorithms. The authors claim that this method surpasses state-of-the-art human-designed and learning-based heuristics in terms of solution quality and efficiency across various benchmarks. However, the novelty of the approach appears limited, as similar uses of LLMs for algorithm design and optimization have already been explored in recent literature. Notably, works such as "Evolution of heuristics: Towards efficient automatic algorithm design using large language model," "LLaMoCo: Instruction tuning of large language models for optimization code generation," "LLaMEA: A large language model evolutionary algorithm for automatically generating metaheuristics," "ReEvo: Large Language Models as Hyper-Heuristics with Reflective Evolution," "Autonomous multi-objective optimization using large language model," and "OptiMUS: Optimization Modeling Using MIP Solvers and large language models" have addressed related ideas. While LLM4Solver demonstrates promising results, the core concept of prompting LLMs to generate code for optimization tasks is not particularly novel, and the paper primarily compares itself to the work of Kuang et al. without sufficiently situating its contributions relative to the broader body of recent work. The main contribution seems to be the specific integration of LLM-generated code with evolutionary refinement, but this incremental advance does not substantially differentiate the approach from existing methods. To more convincingly establish its novelty, the paper would benefit from expanded experimental comparisons and a clearer articulation of how its framework advances beyond prior LLM-based optimization techniques.
