ProX introduces a novel approach to data refinement for language model pre-training by framing it as a program synthesis and execution task, using small language models to generate and apply per-example operations at scale. While the explicit use of program synthesis for data refinement is a substantive innovation, the broader paradigm of per-example, model-driven data curation is already present in recent works such as QuRating and MATES. The authors’ claims of firstness are accurate for the programming aspect, but somewhat overstated regarding per-example, model-driven refinement, which is a shared trend in the field. Empirical improvements over prior methods are demonstrated, but it is unclear how much these gains stem from the programming paradigm versus implementation details or hyperparameter choices. Reviewers should note that some relevant related work (e.g., RegMix, Platypus) is omitted, and that the main contribution is an operational advance rather than a conceptual shift in the field.
