This paper introduces Tool Decoding, a training-free, plug-and-play approach designed to enhance large language models’ (LLMs) tool-use capabilities by integrating tool-specific data directly into the decoding phase. The method leverages constrained decoding to improve format accuracy and an order consistency mechanism to ensure parameter correctness, aiming to reduce common tool-use errors such as key, value, and format errors. However, the novelty of the approach is limited, as constrained decoding for tool usage in LLMs has already been explored in several prior works, including Domino (which focuses on grammar and token alignment in general constrained text generation tasks) and TOOLDEC (which eliminates syntax errors by constraining token choices using finite state machines to maintain tool syntax). The specific implementation of constrained decoding in this paper does not differ significantly from these existing methods. Overall, while the paper presents a practical and adaptable solution, its core techniques largely build upon established approaches, and the incremental advance provided by order consistency does not substantially advance the state of the art.
