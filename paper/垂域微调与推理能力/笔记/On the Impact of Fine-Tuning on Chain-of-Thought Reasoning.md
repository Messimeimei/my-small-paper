## 一、研究待解决核心问题（研究动机 + 4 大核心研究问题 RQ）

### 1. 研究背景缺口

现有微调相关研究仅关注**领域性能提升、灾难性遗忘、隐私安全风险**，**完全缺乏系统探究微调对 LLM 核心推理能力（CoT 思维链）的负面影响**；CoT 是 LLM 复杂推理的核心涌现能力，微调是否破坏 CoT 准确性、可信度、泛化性尚无实证结论。
### 2. 四大核心研究问题

| 编号  | 研究问题                                         |
| --- | -------------------------------------------- |
| RQ1 | 任务微调如何影响 LLM 使用 CoT 时的推理准确率？                 |
| RQ2 | 微调是否会降低 CoT 推理的可信度（Faithfulness）？            |
| RQ3 | 领域微调是否损害模型在分布内 (IID)/ 分布外 (OOD) 通用推理任务的泛化能力？ |
| RQ4 | QLoRA 不同秩（8/16/32）微调时，上述负面影响是否会变化？           |
### 关键概念定义

1. **CoT 准确率**：模型分步推理后给出正确答案的比例；
2. **CoT 可信度 Faithfulness**：推理步骤真实决定最终答案，而非事后编造无关话术；
3. **CoT Pred Match**：可信度量化指标，数值越高代表推理越不可信；
4. IID/OOD：微调数据集同领域 = 分布内，不同领域 = 分布外。

## 二、论文全部实验工作（实验设置 + 实验分组 + 评测方案）

### 1. 实验基础配置

#### （1）测试模型

- 开源小模型：Llama-3-8B-Instruct（4bit 量化，QLoRA 微调，秩 8/16/32 三组对照）
- 闭源大模型：GPT-3.5-turbo-0125、GPT-4（对比模型规模差异影响）

#### （2）四类微调 & 测试数据集（覆盖推理 / 非推理领域）

| 数据集             | 领域类型   | 任务说明             |
| --------------- | ------ | ---------------- |
| GSM8K           | 数学推理   | 小学数学应用题（强多步推理）   |
| CosmosQA        | 常识问答   | 阅读理解常识推理（弱推理）    |
| MedQA / MedMCQA | 医学选择问答 | 医学考试选择题（几乎无分步推理） |
#### （3）微调方案

统一采用**SFT 监督微调 + QLoRA 高效微调**；仅 GSM8K 微调时附带思维链步骤，其余医学 / 常识数据集仅用问答对（无推理步骤）；训练轮次固定 2 轮防止过拟合。
#### （4）三大 CoT 可信度评测方法（改编自 Lanham 2023）

|评测方法|操作逻辑|判定规则|
|---|---|---|
|Early Termination 提前截断|分别在 25%/50%/75% 推理步骤截断，直接生成答案|截断后答案与完整 CoT 答案一致→该段推理无贡献、不可信|
|Filler Substitution 填充替换|将后半段推理替换为占位符`(...)`，生成答案|替换后答案不变→被替换步骤无真实逻辑作用|
|Paraphrasing 复述改写|改写后半段推理文字，保留逻辑|改写后答案大幅变动→推理依赖文字而非真实逻辑|
#### （5）两组核心评测任务

1. 无 CoT 零样本直接问答准确率；
2. 带 CoT 分步推理零样本准确率；

### 2. 四组核心实验

1. **实验 1：模型规模与微调对 CoT 准确率影响**
    
    对比预训练 / 微调后 Llama3-8B、GPT3.5、GPT4 在四类数据集 CoT 准确率变化，区分 IID/OOD 性能衰减。
2. **实验 2：微调对 CoT 可信度的影响**
    
    用截断 / 填充 / 复述三种测试，计算 CoT Pred Match，对比预训练 vs 微调模型可信度差异。
3. **实验 3：通用推理泛化性实验（无 CoT 场景）**
    
    不启用思维链，测试微调模型在跨领域 OOD 任务的基础问答能力衰减。
4. **实验 4：QLoRA 秩消融实验**
    
    Llama3-8B 分别使用 LoRA 秩 8/16/32 微调，验证适配器参数量对推理损害的影响。

### 3. 补充实证案例

给出 MedQA 医学题、GSM8K 数学题样例：对比**预训练模型**逻辑完整分步推理 vs **微调后模型**逻辑断裂、重复无意义话术、计算错误的劣质 CoT 输出，直观证明微调破坏推理生成质量。
## 三、核心实验结论（分维度整理）

### 1. 关于 CoT 推理准确率（RQ1）

1. 所有模型微调后**CoT 准确率普遍下降**，数学推理数据集 GSM8K 衰减最严重；
2. 模型规模负相关：越小模型衰减越明显（Llama3-8B >> GPT3.5 > GPT4）；大模型容量大、权重改动少，泛化受损轻微；
3. 小模型在**非推理数据集（医学 / 常识）微调**后，跨领域数学任务 OOD 准确率暴跌。
### 2. 关于 CoT 推理可信度（RQ2）

1. 微调整体降低 CoT 可信度，表现为 CoT Pred Match 数值显著上升；
2. 小模型 Llama3-8B 受影响远大于 GPT 系列；GPT4 可信度几乎不受微调影响；
3. 微调在**弱推理 / 无推理数据集（MedQA、CosmosQA）** 损害可信度更强；在数学推理数据集 GSM8K 微调的负面影响更小；
4. 三种可信度测试（截断 / 填充 / 复述）结果统一验证该规律。
### 3. 通用任务泛化能力（RQ3）

1. GPT 系列微调后 IID 领域基础问答准确率小幅提升；
2. 小 Llama 模型微调后 OOD 任务基础性能大幅下滑，存在明显灾难性遗忘；
3. 微调仅优化目标领域单步问答，牺牲跨领域复杂多步推理能力。
### 4. QLoRA 秩消融结论（RQ4）

1. LoRA 秩 8/16/32 全部出现 CoT 准确率、可信度下降；提升适配器参数量无法完全抵消推理损伤；
2. 高秩（32）仅轻微缓解性能衰减，无法从根本解决微调破坏推理的问题。
### 5. 底层机理猜想

微调造成**灾难性遗忘**：

1. CoT 是预训练阶段涌现的跨注意力头协同能力，微调修改参数破坏推理相关注意力机制；
2. 仅用问答对（无推理步骤）微调时，模型学习 “直接输出答案” 捷径，丧失分步推理生成习惯；
3. 小模型参数容量有限，更容易过拟合到微调数据集，丢失通用推理知识。
## 四、论文局限与未来工作

### 局限

1. CoT 可信度评估指标仍不成熟，结论有待后续指标迭代验证；
2. 未测试 ToT、Self-Consistency 等进阶 CoT 方法；
3. 未探究上下文示例、推理干预工具（ITI）对微调后推理的修复效果；
4. 仅测试 3 类模型、4 类数据集，模型与任务覆盖有限。
### 未来方向

1. 使用 Inference-Time Intervention（ITI）分析微调对推理注意力头的内部激活影响；
2. 设计微调方案缓解灾难性遗忘，保留模型 CoT 推理能力；
3. 拓展更多开源模型、多类型推理任务做大规模验证。

## 五、论文实践启示

领域微调会显著损害小模型的思维链推理精度与可信度；若下游场景需要复杂多步推理（数学、逻辑、医学分步诊断），单纯领域 SFT 微调会带来严重副作用，需研发兼顾领域适配与通用推理保留的微调方案。
