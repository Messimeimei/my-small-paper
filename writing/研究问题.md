# 研究问题

本文件只维护“正在回答什么、已有证据是什么、下一步怎样证伪”。实验数字统一引用 [实验记录.md](实验记录.md)，不在这里重复完整表格。

状态含义：`已回答`、`初步证据`、`未回答`、`暂停`。只有经过公平对照和重复实验的问题才能标为“已回答”。

## 1. 当前主问题

> 小模型能否学习并组合科学写作评价中的可复用推理能力，在训练见过和未见的科学文本任务上都取得提升，同时减少按任务独立微调产生的能力干扰？

这里包含两层假设：

- 科学假设：科学写作评价可以分解为证据验证、定位归因、标准遵循、比较判断、不确定性估计等能力，并通过统一的“证据状态”接口迁移。
- 工程假设：多个 LoRA 专家及动态路由可以承载并组合这些能力；MoDE / Self-MoE 是候选实现，不是论文贡献本身。

当前状态：`未回答`。现有结果只证明单个 coherence 专家能在部分 Related Work aspect 上改变性能，还没有证明“推理原子可组合”或“OOD 迁移”。

## 5. 当前优先级

1. 先修复实验身份：准确评测 DeepSeek checkpoint 205 / 410 / 615，保证目录名、`exp_name` 和 adapter 一致。
2. 在共同样本上完成 DeepSeek、GLM、双教师共识的公平单专家对照，同时补 label-only / answer-only baseline。
3. 形成单专家在 Seen、Unseen 和外部任务上的完整迁移矩阵，再决定是否值得进入多专家路由。

在前两步完成前，“GLM 教师优于 DeepSeek”以及“推理原子可以组合迁移”都只能作为假设，不能写成结论。
