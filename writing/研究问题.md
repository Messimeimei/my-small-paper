# 研究问题

更新时间：2026-07-24。

本文件只维护“论文究竟要回答什么、什么证据才算回答、什么结果会推翻主张”。实验事实统一引用 [实验记录.md](实验记录.md)，执行顺序统一引用 [科研流程.md](科研流程.md)。

本轮研究审查接受一个前提：训练集、校准集和测试集的独立性已经确认，不再把数据重合问题作为研究路线的争议点。下面只审查研究构念、方法因果链、对照、公平性、统计稳健性和贡献边界。

## 1. 研究契约

### 1.1 研究对象

本项目研究的是：

> **Rubric-conditioned scientific writing evaluation**：给定科学文本、评价准则、评分空间和示例，让小模型生成有依据的评价过程与最终分数。

当前不把它泛化表述为“科学领域复杂推理”。Related Work、Review Utility、Novelty 和 Revision 能支撑的是不同准则下的科学写作评价；`meta_reviewer` 暂时只是复杂压力测试，不承担主论文成败。

### 1.2 当前主问题

> 在固定目标监督预算下，由不同科学写作评价准则训练得到的 CoT-LoRA 专家，能否被可靠选择或组合，并在新评价准则上稳定优于最佳单专家、简单融合和同预算目标适配？如果能，收益是否确实来自 CoT 监督与跨专家互补，而不是单专家能力或少量目标监督本身？

这比原问题更窄，但与当前实现一致：当前 MoDE 为每个目标任务使用少量有标签 calibration 样本搜索一组**任务级全局权重**。它是 `new-target few-shot adapter adaptation`，不是 zero-shot 泛化，也不是样本级动态路由。

### 1.3 当前术语约束

| 术语 | 当前允许的含义 | 升级条件 |
|---|---|---|
| 准则专家 / 任务专家 | 在某个 aspect 的 CoT 监督上训练的 LoRA | 当前即可使用 |
| 可复用评价能力 | 在预注册的相关异任务上出现可预测正迁移 | 完整 expert x target 矩阵及负对照成立 |
| 推理原子 | 能跨表面任务复用、具有操作化定义和可观测中间状态的能力 | RQ2 通过后才使用 |
| 专家组合 | 多专家方法超过最佳单专家和简单组合 | RQ3 通过后才使用 |
| 少样本新目标适配 | 使用目标任务 K-shot supervision 搜索或训练 | 当前 MoDE 的准确称呼 |
| 动态路由 | 权重随输入样本变化 | 当前未实现，不用于描述现有方法 |
| 减少能力干扰 | 一个固定部署策略在 retention matrix 上优于联合微调 | RQ6 通过后才使用 |

## 2. 因果链与研究问题

研究问题必须按下面的顺序回答，不能从“相对 base 提升”直接跳到“推理原子可组合”：

```text
CoT 是否有因果价值
  -> 专家是否形成可预测的跨任务复用结构
  -> 组合是否优于选择与简单融合
  -> 是否优于同监督预算的目标适配
  -> 组合机制是否数学稳健
  -> 才能讨论保持能力、效率和更广泛泛化
```

| ID | 研究问题与可证伪预测 | 决定性实验 | 失败后的处理 | 当前状态 |
|---|---|---|---|---|
| RQ1 能力习得 | Full-CoT 在相同样本、更新步数和 LoRA 配置下，应稳定优于 answer-only / score-only；无关等长 rationale 不应复制收益 | `score-only / answer-only / full-CoT / shuffled-or-irrelevant rationale`，至少 3 个训练 seed | 停止“CoT 蒸馏提升 reasoning”，改称普通任务适配 | `未回答` |
| RQ2 能力复用 | 与目标所需能力相关的专家，应在非本任务上系统优于无关专家和 base | 完整 expert x target 迁移矩阵；预先写明正向迁移对和 negative control | 若只有对角线提升，统一称 task adapter，不称推理原子 | `未回答` |
| RQ3 组合价值 | 在相同 K-shot calibration 下，组合应稳定超过 calibration-best-single、`c * one-hot`、`c * equal` 和随机 mix | 在主 transfer targets 上比较 scalar controls、factor / delta 两类 searched 与 simple baselines，至少 10 个 calibration seed | 若只优于 base，不再声称协同，转为专家选择或强度校准 | `当前不支持` |
| RQ4 监督归因 | 在相同目标监督与计算预算下，组合应优于 ICL、K-shot target-only LoRA 和 joint multi-task LoRA | 固定 K、样本、搜索/更新预算和评测协议 | 若不占优，收益归因于目标监督，不归因于专家复用 | `未回答` |
| RQ5 机制有效性 | 等价的 LoRA rank permutation / rescaling 不应改变方法结论；搜索目标下降应对应最终评分改善；删除相关专家的损失应大于删除无关专家 | factor mix vs delta-sum、rank permutation / rotation、A/B rescaling、cross-term、leave-one-out，以及 score-only / example-macro / token-micro CoT 目标对照 | 若 factor mix 坐标敏感，禁止解释权重，转向 delta-space 或显式对齐；若目标错位，改用 score-aligned objective | `未回答` |
| RQ6 保持能力 | 一个固定部署策略应在多任务 retention matrix 上优于 joint fine-tune，并控制最差任务回退 | 固定组合或固定路由的全任务矩阵，不允许每个测试任务单独重搜后再称“一个模型” | 若不实现固定策略，从论文中删除“减少干扰”主张 | `未回答` |
| RQ7 稳健性与效率 | 主效应应跨训练 seed、calibration seed 和 k=1/3/5 保持，并具有合理成本 | 3 个训练 seed、10 个 calibration seed、k 曲线、GPU-hours / 搜索调用 / 推理开销 | 不稳定则收窄 claim；成本不占优则删除效率贡献 | `未回答` |

## 3. 当前证据判决

| 候选主张 | 当前证据 | 判决 | 当前可写表述 |
|---|---|---|---|
| 单任务 LoRA 能改善部分评价任务 | actionability 单专家由 `.3016/.2108` 提升到 `.4914/.4442`（Accuracy / macro-F1） | `初步支持` | 任务特定 LoRA 在部分准则上有明显收益 |
| target-specific factor mix 相对 base 有收益 | novelty、revision relatedness / correctness 及三个已有专家任务均有描述性提升 | `描述性支持` | 少量目标监督校准后的 factor mix 改善若干任务 |
| 多专家组合优于单专家 | 三个 seen target 相对对应单专家的 Accuracy 为 `+0.06/-0.40/-1.47 pp` | `当前不支持` | seen target 主要表现为专家选择或强度缩放 |
| 专家发生跨任务协作 | transfer targets 尚缺全体单专家、best-single 和 leave-one-out | `未回答` | 只能作为 RQ，不能写成结论 |
| CoT 是收益来源 | 尚无 answer-only、score-only 和无关 rationale 对照 | `未回答` | 使用“CoT-supervised adapter”，不写 reasoning 已增强 |
| 推理原子可组合 | 当前专家按 actionability、positioning_check/type 等任务 aspect 构造 | `未操作化` | 使用 criterion/task expert |
| zero-shot 泛化 | 每个新目标使用目标监督 calibration | `不适用` | 使用 new-target few-shot adaptation |
| 减少能力干扰 | 没有固定模型的 retention matrix | `未测试` | 暂时移出主问题 |
| factor 权重可解释能力贡献 | A/B 分别混合产生交叉项和全局符号对称性 | `不成立` | 用 leave-one-out 或 Shapley-style 性能贡献，不解释权重正负 |
| 复杂科学推理增强 | 当前主任务是准则判断，meta_reviewer 尚未形成有效性能 | `不支持` | 限定为科学写作评价 |
| 方法具有独立新颖性 | MoDE 与任务框架均来自已有工作 | `当前不足` | 先做严格实证；只有解决新机制问题后升级方法贡献 |

## 4. 两篇来源论文与本项目的关系

- MoDE-CoTD 提供“分别蒸馏任务 LoRA，再用少量目标样本搜索组合系数”的技术起点。它已经主张 seen / unseen 任务上的跨任务协作，因此“把 MoDE 用到科学写作”本身不构成方法创新。
- Reward Modeling for Scientific Writing Evaluation 提供以 evaluation constitution、动态准则和多 aspect 联合训练为核心的任务框架，并已经报告 unified reward model 的 seen / unseen 泛化。它是本项目最重要的 `joint multi-task` 强基线，而不只是数据来源。该论文移除推理提示的 inference-time ablation 不能替代本项目的 full-CoT / answer-only 训练因果对照。
- 本项目只有在下列至少一项成立时，才能形成独立贡献：证明准则专家存在可预测的组合结构；提出并验证更稳健的 LoRA 组合机制；或系统刻画 modular adapters 相对联合训练的收益边界与成本。

## 5. 论文路线

当前只选一条主线，避免同时背负“推理原子、动态路由、OOD、抗遗忘和低成本”五套证据责任。

1. **默认路线：严格实证论文。** 研究 CoT 准则专家在科学写作评价中的少样本选择、组合与迁移边界，先完成 RQ1-RQ4。
2. **升级路线：方法论文。** 只有 RQ5 暴露出可复现的 factor-mix 缺陷，并且新方法稳定解决该缺陷且通过 RQ3，才转为 coordinate-invariant composition 或显式 rank alignment。
3. **保底路线：应用论文。** 若组合不优于选择，但模块化专家在维护、扩展或成本上有清晰优势，则定位为高效多准则评价系统。

## 6. 当前优先级与止损线

1. 完成三个现有专家在主任务上的完整迁移矩阵，不再先训练更多专家。
2. 在 `revision_relatedness` 和 `revision_correctness` 上完成 `best-single / scaled-single / scaled-equal / equal-delta / searched-factor / searched-delta` 决战。
3. 在 actionability 上完成 CoT 因果消融，再决定是否扩到第二个专家。
4. 对 factor mix 做 rank permutation、A/B rescaling、delta-sum 和搜索目标对齐实验。
5. 只有通过上述筛选的方法，才扩展到 k=1/3/5、多 seed、retention 和成本实验。

硬停止条件：

- searched composition 若不能跨 calibration seeds 稳定超过 best-scaled-single、scaled-equal 和 delta-sum，停止“多专家协同”主张，也停止增加复杂路由。
- full-CoT 若不能跨训练 seeds 稳定超过 answer-only，停止“推理蒸馏提升 reasoning”主张。
- 若 factor mix 对等价 LoRA 重参数化敏感，停止使用权重热图解释专家能力，优先改用 delta-space 组合。
- `meta_reviewer` 在输出协议达到可用水平前只作压力测试，不围绕它继续扩专家。
